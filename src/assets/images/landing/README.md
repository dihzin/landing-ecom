# Maluz Dashboard — Landing Assets

Esta pasta reúne exclusivamente os prints finais da landing page.

## Padrão de nomes (semântico e orientado a valor)
- hero-dashboard.png
- margin-decision.png
- inventory-giro.png
- team-performance.png

## Diretrizes rápidas
- Substitua os arquivos mantendo o mesmo nome para evitar mudanças no HTML.
- Use imagens recortadas com foco em valor (margem, giro, performance).
- Prefira resoluções consistentes entre os prints para preservar o layout.

## Especificações recomendadas
- Formato: PNG.
- Largura alvo: 1600px (mínimo 1400px).
- Proporção:
  - hero-dashboard.png: 16:9 ou 15:9.
  - margin-decision.png: 4:3 (recorte mais alto).
  - inventory-giro.png: 3:2.
  - team-performance.png: 3:2.
- Peso do arquivo: até 900KB por imagem.
- Nitidez: exportar em 2x (retina) e reduzir para a largura alvo.

## Recorte e conteúdo
- Evite áreas vazias ou bordas de janela do sistema operacional.
- Priorize números e indicadores na área central do recorte.
- Não use sombras ou molduras extras; o layout já aplica estilo.
